# Homepage

A small personal homepage built as part of **Harvard's CS50 – Introduction to Computer Science**, specifically the **HTML, CSS and JavaScript** assignment.

The goal of the project was to create a simple multi-page website introducing myself and my interests while learning the fundamentals of front-end development.

I decided to publish it as a separate repository and treat it as my **first attempt at building a clean-looking personal website**.

---

## Features

- Responsive layout using **Bootstrap 5**
- **Dark / Light mode toggle** with localStorage persistence
- **Live clock** displayed in the navigation bar
- **Simple JS flashcard system** for Mandarin vocabulary
- Multi-page structure with navigation

The dark mode toggle is implemented in JavaScript and stores the user's preference locally.

The Mandarin practice page includes a small flashcard system that randomly cycles through phrases written in **Chinese characters, Zhuyin, and English translation**.
